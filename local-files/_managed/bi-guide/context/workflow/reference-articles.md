# Reference Articles

Bestehende BI-Guide-Artikel sind die primaere Stil- und Strukturreferenz.

Nutze sie fuer:
- Tonalitaet
- Absatzrhythmus
- Tiefe pro Zielgruppe
- Umgang mit Callouts, Listen, Quellen und internen Links

Nicht uebernehmen:
- thematische Redundanz
- gleiche Reihenfolge oder identische Formulierungen
- starre Schablonen ohne fachlichen Grund

Ziel:
- wiedererkennbar Sandbank
- aber nicht monoton
